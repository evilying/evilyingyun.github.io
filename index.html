关于THUThesis Latex出现的错误

今天使用THUThesis模版，编译时出现了诸多问题，将重要的部分列出，以待后续时日查看，如果能给读者以方便之处，当属幸事。

依赖项问题

Use Package listing error: Listing.sty not found

MikTex本来是自动下载依赖包，不过这个style依赖是个意外，解决方法是

  • 输入mpm命令，启动MikTex Package Manager
  • 找到Listing.sty进行安装

字体问题

Trying to make PK font * at &&& DPI

这个错误让人猝不及防，我怀疑是ttf包的问题,就重新下载了对应出问题的ttf，然而并没什么卵用。
之后搜索到解决方案，找到miktex的安装位置,在bin目录下执行
initexmf –mkmaps
initexmf –update-fndb
这个应该是重新更新一下字体库的映射关系。不过这个不是通用的解决方案，我之前遇到了更加棘手的问题，
要自己重新建立映射，具体的忘记了。

make问题

编译一把竟然要13秒，而且还无法自动弹出PDF文件，实在难以忍受。想了想问题所在，之前是pdflatex直接编译就可以了，
为何要他们提供的Makefile，尝试删去，竟然可以了，关键是快外加自动弹PDF :(|)。
估计是因为latex编译器首先执行本地的Makefile，如果找不到，再按照自己的Makefile编译一把。
